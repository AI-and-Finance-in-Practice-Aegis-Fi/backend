# AI Agent Design

## Agent 1: Ghost Account Detector

Purpose:

Detect inactive SaaS users.

Input:

- last_login_date
- monthly_usage_count

Output:

- is_ghost_account
- risk_level

---

## Agent 2: SaaS Cost Optimizer

Purpose:

Identify SaaS cost-saving opportunities.

Input:

- total_seats
- used_seats
- monthly_fee

Output:

- wasted_amount
- recommendation

---

## Agent 3: Transaction Risk Analyzer

Purpose:

Analyze employee spending risk.

Input:

- amount
- category
- user_input_reason

Output:

- ai_risk_score
- ai_risk_reason

---

## Agent 4: AI CFO Agent

Purpose:

Generate executive-level financial insights.

Input:

- SaaS analysis
- transaction analysis
- department budgets

Output:

- recommendations
- CFO summaries
- saving opportunities
